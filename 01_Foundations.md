### AI

AI הוא התחום הרחב. ML הוא תת-תחום. Deep Learning הוא תת-תחום של ML. LLM
הוא מודל Deep Learning לשפה.

### LLM

המטרה של LLM היא לחזות את ה-Token הבא. כל היכולות (צ'אט, קוד, תרגום
וכו') נובעות מכך.

### Tokens

יחידת העבודה של המודל. משפיעה על מחיר, Context, Latency ועלות.

### Context Window

כולל System Prompt, User Prompt, Memory, RAG ו-Tool Results.

### Temperature

נמוך = עקבי. גבוה = יצירתי.

### Streaming

מחזיר תשובה בהדרגה ומשפר UX.

### Hallucinations

המודל עלול להחזיר מידע שנשמע נכון אך אינו נכון. מצמצמים באמצעות RAG,
Tools ו-Validation.

AI Engineer Tips: - שלח רק Context רלוונטי. - אל תסמוך על תשובת המודל.
