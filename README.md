# AI Engineering Handbook

מדריך עיון מקצועי בעברית (RTL) לבניית מערכות AI מבוססות LLM — מהיסודות התיאורטיים ועד לפריסה בפרודקשן.

אתר סטטי, ללא build step, offline-first (כל הפונטים, האייקונים וה-syntax highlighting מאוחסנים מקומית תחת `assets/`).

## פתיחה

פתחו את `index.html` בדפדפן.

## מבנה

- `index.html` — שער הספר ותוכן עניינים
- `chapters/` — 9 פרקי הספר
- `assets/` — עיצוב, סקריפטים, וגופנים/אייקונים מקומיים
- `01_Foundations.md` … `06_Architecture_and_CheatSheet.md` — חומר המקור שעליו התבסס הספר
- `docs/superpowers/` — מסמכי spec ו-plan של תהליך הבנייה
